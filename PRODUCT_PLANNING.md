# 应用中心：

我现在本地搭了一个 agent 团队，两种模式，一种自动挡，一种手动挡。自动挡就是团队自主运行和讨论，自己跑，检查系统状态，需要重大决策了，在飞书群里面 @ 我拍板；手动档就是我直接操控 agent 干活。系统搭完之后我觉得整个人都解放了，比如服务端层的频繁健康探测、日志层的 ops agent 定时检测，检测完了发现问题 转接给其他 agent，变成多 agent 协作的问题
