# Design System Inspired by MiniMax

## 1. Visual Theme & Atmosphere

MiniMax is an AI company with a modern, approachable design language that balances technical sophistication with user-friendly presentation.

**Key Characteristics:**
- Modern AI aesthetic
- Approachable technical design
- Clean, professional layouts
- Balanced color temperature

## 2. Color Palette & Roles

### Primary
- **MiniMax Blue** — Brand accent, CTAs
- **Dark Navy** — Primary text and surfaces
- **Light Gray** — Background surfaces

### Secondary
- Warm accent colors
- Technical neutral palette

## 3. Typography Rules

### Font Family
- **Primary**: Modern geometric sans-serif
- **Body**: Clean, readable typeface

### Hierarchy
| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Display | 64px | 600 | 1.10 |
| Section | 40px | 600 | 1.20 |
| Body | 16px | 400 | 1.50 |

### Principles
- Modern weight distribution
- Clean typographic hierarchy
- Readable body text

## 4. Component Stylings

### Buttons
**Primary Blue**
- Background: MiniMax Blue
- Radius: 8px
- Hover: Slightly darker

### Cards
- Clean white backgrounds
- Subtle shadows
- 12px radius

## 5. Layout Principles

- Generous section spacing (64px+)
- Modern grid layouts
- Centered content blocks

## 6. Depth & Elevation

- Subtle card shadows
- Layered surfaces
- Professional depth

## 7. Do's and Don'ts

### Do
- Use modern, clean typography
- Maintain generous whitespace
- Keep layouts balanced

### Don't
- Add heavy decorative elements
- Overcrowd content areas
