# Design System Inspired by Superhuman

## 1. Visual Theme & Atmosphere

Superhuman is an email client focused on speed and productivity, with a premium, minimalist design that emphasizes efficiency and elegance. The visual language communicates "get through your inbox faster" through clean lines, dark surfaces, and a restrained palette.

**Key Characteristics:**
- Premium, productivity-focused aesthetic
- Dark mode by default
- Speed-optimized interface
- Minimalist, efficient layouts
- Keyboard-first design philosophy

## 2. Color Palette & Roles

### Primary
- **Superhuman Dark** — Primary surface
- **Superhuman Blue** — Accent, CTAs
- **Pure White** — Text on dark surfaces

### Secondary
- Subtle accent variations
- Semantic status colors
- Neutral grays for hierarchy

## 3. Typography Rules

### Font Family
- **Primary**: Clean geometric sans-serif
- **Mono**: Technical content, shortcuts

### Hierarchy
| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Display | 48px | 600 | 1.10 |
| Section | 32px | 500 | 1.20 |
| Body | 16px | 400 | 1.50 |

### Principles
- Weight 500-600 for display
- Tight line-heights for headlines
- Relaxed reading for body text

## 4. Component Stylings

### Buttons
**Primary Blue**
- Background: Superhuman Blue
- Radius: 6px (functional)
- Hover: Brightened shade

### Cards
- Dark surfaces by default
- Minimal borders
- Efficient content layout

### Keyboard Shortcuts
- Prominently displayed
- Monospace font
- Subtle backgrounds

## 5. Layout Principles

- Efficiency-focused spacing
- Information density optimized
- Keyboard-first navigation

## 6. Depth & Elevation

- Minimal shadows
- Surface layering for hierarchy
- Dark mode depth through contrast

## 7. Do's and Don'ts

### Do
- Emphasize keyboard shortcuts
- Maintain dark surfaces
- Optimize for speed reading

### Don't
- Add decorative flourishes
- Use light mode by default
- Slow down the interface
