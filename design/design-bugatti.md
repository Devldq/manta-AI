# Design System Inspired by Bugatti

## 1. Visual Theme & Atmosphere

Bugatti's website is a hypercar brand with a cinematic black canvas and monochrome strictness. The entire experience lives on Velvet Black (`#000000`), with only Showroom White (`#ffffff`) for text and accents.

**Key Characteristics:**
- Velvet Black (`#000000`) — entire canvas
- Showroom White (`#ffffff`) — all text, borders, CTAs
- No accent colors — products provide color
- 288px display headlines — monumental, architectural
- 9999px pill buttons — maximum rounding
- Zero card components — full-bleed media blocks
- No bold weights — scale carries hierarchy

## 2. Color Palette & Roles

### Primary
- **Velvet Black** (`#000000`): Entire canvas
- **Showroom White** (`#ffffff`): All text, borders, CTAs
- **Silver Mist** (`#999999`): Only gray, disabled states

### Principles
- Strict monochrome — no exceptions
- Products introduce color, UI stays neutral

## 3. Typography Rules

### Font Family
- **Display**: `Bugatti Display` — proprietary
- **Monospace**: `Bugatti Monospace` — UI labels
- **Body**: `Bugatti Text Regular`

### Hierarchy
| Role | Size | Weight | Letter Spacing |
|------|------|--------|----------------|
| Hero Display | 288px | 400 | normal |
| Section | 80px | 400 | normal |
| Body | 16px | 400 | normal |
| Mono Label | 12px | 400 | 1.2-1.4px |

### Principles
- No bold weights — scale IS hierarchy
- Monospace ALWAYS uppercase with positive tracking
- Tight line-heights for display

## 4. Component Stylings

### Buttons
**White Pill**
- Background: `#ffffff`
- Text: `#000000`
- Radius: 9999px — full pill

### No Cards
- Full-bleed media blocks
- No container components

## 5. Layout Principles

- Full-bleed sections
- Zero padding on media
- Minimal UI chrome

## 6. Depth & Elevation

- No shadows
- No borders on containers
- Flat, cinematic presentation

## 7. Do's and Don'ts

### Do
- Use ONLY black and white
- Let products provide color
- Scale type for hierarchy

### Don't
- Introduce accent colors
- Use card components
- Apply bold weights
