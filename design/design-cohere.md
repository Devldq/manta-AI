# Design System Inspired by Cohere

## 1. Visual Theme & Atmosphere

Cohere's website presents a sophisticated AI platform aesthetic with a clean, modern design language. The interface emphasizes clarity and accessibility for an AI/ML developer audience.

**Key Characteristics:**
- Clean, professional appearance suited for enterprise AI
- Balanced use of whitespace for focus and readability
- Typography hierarchy that guides developer attention

## 2. Color Palette & Roles

### Primary
- **Cohere Blue** — Primary brand and interactive accent
- **Dark Navy** — Text and dark surfaces
- **White** — Primary surface

### Semantic
- Success, warning, error states follow standard patterns

## 3. Typography Rules

- Modern geometric sans-serif for clarity
- Clear hierarchy from display to body text
- Code-friendly monospace for technical content

## 4. Component Stylings

- Clean button treatments with appropriate hierarchy
- Card-based content organization
- Form inputs optimized for developer workflows

## 5. Layout Principles

- Generous section spacing
- Responsive grid systems
- Content-first approach

## 6. Depth & Elevation

- Subtle shadows for layered interfaces
- Border-based containment where appropriate

## 7. Do's and Don'ts

### Do
- Maintain clean, professional appearance
- Use consistent spacing throughout

### Don't
- Overcrowd interfaces with excessive visual elements
