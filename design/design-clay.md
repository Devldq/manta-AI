# Design System Inspired by Clay

## 1. Visual Theme & Atmosphere

Clay's design represents a modern business tool aesthetic with emphasis on flexibility and professional workflows.

**Key Characteristics:**
- Professional, clean interface
- Flexible component system
- Enterprise-appropriate design language

## 2. Color Palette

### Primary
- Neutral business palette
- Accent colors for key actions
- Surface variations for depth

## 3. Typography

- Professional sans-serif family
- Clear typographic hierarchy
- Readable body text

## 4. Components

- Flexible card systems
- Data presentation elements
- Navigation patterns

## 5. Layout

- Grid-based content organization
- Responsive breakpoints
- Consistent spacing system

## 6. Depth

- Subtle elevation through shadows and borders
- Surface layering for visual hierarchy

## 7. Guidelines

### Do
- Maintain professional appearance
- Use consistent spacing

### Don't
- Over-decorate functional elements
