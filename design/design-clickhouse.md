# Design System Inspired by ClickHouse

## 1. Visual Theme & Atmosphere

ClickHouse is a high-performance columnar database company with a technical, developer-focused design language. The visual approach emphasizes speed, efficiency, and data processing power.

**Key Characteristics:**
- Technical, data-focused aesthetic
- High-performance messaging
- Developer-centric design language
- Clean, efficient layouts

## 2. Color Palette & Roles

### Primary
- **ClickHouse Yellow** — Brand accent
- **Deep Navy** — Primary dark surfaces
- **Pure White** — Primary light surface

### Secondary
- Performance-oriented accent colors
- Technical neutral palette

## 3. Typography Rules

### Font Family
- **Primary**: Technical sans-serif
- **Mono**: Code and technical content

### Hierarchy
| Role | Size | Weight |
|------|------|--------|
| Display | 56px | 700 |
| Section | 36px | 600 |
| Body | 16px | 400 |

## 4. Component Stylings

### Buttons
**Primary Yellow**
- Background: ClickHouse Yellow
- Radius: 6px
- Hover: Darkened shade

### Cards
- Clean, minimal borders
- Technical precision
- Data-focused layouts

## 5. Layout Principles

- Performance-focused spacing
- Technical documentation style
- Developer-first hierarchy

## 6. Depth & Elevation

- Minimal shadows
- Border-based containment
- Flat, efficient design

## 7. Do's and Don'ts

### Do
- Emphasize performance metrics
- Use technical typography
- Maintain clean data layouts

### Don't
- Over-decorate functional surfaces
- Add unnecessary visual complexity
