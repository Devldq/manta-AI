# Plan: Install gstack skill from GitHub

## What is gstack
gstack is Garry Tan's 23-skill toolkit for AI coding agents (CEO reviews, design reviews, QA, code review, etc.). It has native Codex support via `./setup --host codex`.

## Prerequisites
- **bun** (v1.0+) — required by the setup script. Need to verify it's installed.

## Installation Steps

1. **Check bun availability**
   ```bash
   command -v bun && bun --version
   ```

2. **Clone the repo** to `~/.gstack/repos/gstack` (the standard location per setup script migration logic)
   ```bash
   git clone --single-branch --depth 1 https://github.com/garrytan/gstack.git ~/.gstack/repos/gstack
   ```

3. **Run the setup script** targeting Codex
   ```bash
   cd ~/.gstack/repos/gstack && ./setup --host codex
   ```
   This will:
   - Build the browse binary (TypeScript → compiled binary)
   - Install Playwright Chromium for browser-based QA
   - Generate Codex-format skill docs in `.agents/skills/gstack-*`
   - Symlink individual skills into `~/.codex/skills/`
   - Install coreutils on macOS for hang protection

4. **Verify installation**
   ```bash
   ls ~/.codex/skills/gstack-*
   ```

## Notes
- gstack skills are primarily designed for Claude Code; Codex support may have limitations for some skills
- The browse binary build requires network access (npm install)
- Playwright Chromium install requires network access

## Verification
- After setup, check that `~/.codex/skills/` contains gstack-* directories with SKILL.md files
- Restart MiMoCode to pick up new skills
